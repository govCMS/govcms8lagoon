  * clamav 8.x-1.1 with patch: https://www.drupal.org/files/issues/2020-07-21/drupal9_compatibility-3131448-8.patch
  * fast_404 8.x-2.0-alpha5: https://www.drupal.org/files/issues/2020-03-17/3042976-13.patch
